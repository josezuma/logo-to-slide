# FAQ

## What is logo-to-slide?
CLI that places a logo on all 12 slides at correct position, size, and alignment.

## How do I install it?
```bash
git clone https://github.com/josezuma/logo-to-slide.git && cd logo-to-slide
```

## Is this compatible with yc-pitch-deck-max?
Yes. logo-to-slide integrates directly with yc-pitch-deck-max.

## Do I need any external dependencies?
No. All scripts use only Python stdlib.
